Device Selection by URL Pattern

Please set 'media.setsinkid.enabled' to 'true' in 'about:config'!

This is an experimental feature that may not work stably. Also, due to security restrictions, there is no direct access to devices and no ability to always directly set the selected device. Therefore, the user should be prompted once to select a device for a tab. Additionally, this feature may not work correctly with devices that are dynamically connected.
